<img width="1904" height="943" alt="image" src="https://github.com/user-attachments/assets/54081213-cc61-48b6-a11e-bdbdc7eaae51" /># Nearby Places Finder App

A web application to find nearby hospitals, restaurants, and ATMs using location-based services.

## 🔧 Features
- Get user's current location using GPS
- Display nearby hospitals, restaurants, and ATMs
- Interactive map integration
- Responsive design for all devices

## 💻 Technologies Used
- HTML
- CSS
- JavaScript
- Google Maps API

## 📌 How to Run
1. Open index.html in browser
2. Allow location access
3. View nearby places on map
## 🚀 Future Improvements
- Add login/signup authentication
- Save favorite places
